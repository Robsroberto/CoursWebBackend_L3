# # 1. Introduction générale

Ce module a un objectif simple : **Vous donner les bases solides en POO et en architecture MVC** afin que vous soyez opérationnels pour comprendre Laravel.

Laravel **n’est pas magique**.
Il repose sur :

* des **classes** (modèles, contrôleurs, middlewares…)

Ce module vous faire **reconstruire manuellement** ce que Laravel automatise.

# # 2. Rappel fondamental : qu’est-ce qu’un programme orienté objet ?

La POO repose sur quatre notions essentielles :

### 1) **Classes**

Plan de construction d’un objet.
Métaphore : un plan d’une maison → permet de construire plusieurs maisons.

### 2) **Objets**

Instances concrètes de la classe.
Métaphore : les maisons réellement construites.

### 3) **Attributs (propriétés)**

Les données appartenant à l’objet.
Métaphore : couleur, nombre de pièces.

### 4) **Méthodes**

Les actions de l’objet.
Métaphore : ouvrir une porte, allumer la lumière.

# # 3. Exemple simple d’une classe

```php
class Person {
    public $name;

    public function speak() {
        return "Bonjour, je m'appelle " . $this->name;
    }
}

$p = new Person();
$p->name = "Awa";
echo $p->speak();
```

### Explication simple

* `class` = modèle
* `new Person()` = création d’un objet
* `$this->name` = accéder à un attribut interne
* `speak()` = action de l’objet

# # 9. Autoloading : indispensable avant Laravel

Laravel utilise **Composer autoload**.
Nous allons créer **un autoloader manuel**.

### Exemple

```php
spl_autoload_register(function ($class) {
    require_once __DIR__ . "/../app/$class.php";
});
```

# # 10. Architecture MVC – Préparation essentielle pour Laravel

```
project/
│
├── public/
│    └── index.php      # point d'entrée
│
├── app/
│   ├── Controllers/
│   ├── Models/
│   └── Core/           # autoloader + Database
│
└── views/
    └── services/
```

### Pourquoi MVC ?

* **Model** = accéder aux données (comme Eloquent)
* **View** = HTML (comme Blade)
* **Controller** = logique (comme dans Laravel)

# # 11. Workflow complet du projet (schéma explicatif)

### **Utilisateur → Navigue sur /services**

1. La requête arrive dans `public/index.php`
2. Le routeur lit l’URL → appelle `ServiceController@index`
3. Le contrôleur appelle `Service::all()`
4. Le modèle interroge la DB via `PDO`
5. La vue affiche les résultats

### Schéma (texte)

```
Navigateur → Route → Controller → Model → Database
                                     ↓
                                 Retour données
                          Controller → View → HTML
```

C’est **exactement** ce que Laravel fait.

## ✔ 2. Création du **routeur**

<?php

$uri = $_SERVER['REQUEST_URI'];

if ($uri === "/" || $uri === "/services") {
    $controller = new ServiceController();
    $controller->index();

## ✔ 4. Connexion à la base de données

### `app/Core/Database.php`

```php
<?php

class Database {
    private static $pdo;

    public static function connect() {
        if (!self::$pdo) {
            self::$pdo = new PDO("mysql:host=localhost;dbname=mini", "root", "");
            self::$pdo->setAttribute(PDO::ATTR_ERRMODE, PDO::ERRMODE_EXCEPTION);
        }
        return self::$pdo;
    }
}
```

## ✔ 5. Le modèle Service (équivalent Eloquent)

### `app/Models/Service.php`

```php
<?php

class Service {
    public $id;
    public $title;
    public $description;
    public $price;

    public static function all() {
        $db = Database::connect();
        return $db->query("SELECT * FROM services")
                  ->fetchAll(PDO::FETCH_CLASS, Service::class);
    }

    public function save() {
        $db = Database::connect();
        $stmt = $db->prepare("INSERT INTO services (title, description, price) VALUES (?,?,?)");
        return $stmt->execute([$this->title, $this->description, $this->price]);
    }
}
```

---

## ✔ 6. Le contrôleur

### `app/Controllers/ServiceController.php`

```php
<?php

class ServiceController {

    public function index() {
        $services = Service::all();
        require "../views/services/index.php";
    }

    public function create() {
        require "../views/services/create.php";
    }

    public function store() {
        $service = new Service();
        $service->title = $_POST['title'];
        $service->description = $_POST['description'];
        $service->price = $_POST['price'];
        $service->save();
        header("Location: /");
    }
}
```
